# PDF-clipper
  ## package installation
```bash
pip install opencv-python
```
``` bash 
pip install pymupdf
```
  ## run programs
``` bash 
cd pdf-clipper
```
``` bash 
cd python app.py
```
  # outcomes
- feed input file as document.pdf in src folder
- run programs
- select area of text field then press "space" or "enter"
- output saved as cropped_field.jpg in src folder
