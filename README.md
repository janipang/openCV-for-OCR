# docs-head-reader
  ## package installation
```bash
pip install opencv-python pymupdf
```
  ## run programs
``` bash 
cd docs-head-reader
```
``` bash 
cd python app.py
```
  ## outcomes
- feed input pdf files in src/raw-file folder
- run program
- output saved at src/temp/{document_type}/lines
